Getting help and giving feedback can all be done using our User Voice system. Please use the following guide to determine where to get help and provide feedback.

Providing Feedback

If you have an idea for PrePurchasing, please submit them to feedback. There you will be able to browse other user's ideas and vote on the ones you like.

Getting Help for a Specific Issue

Still can't find your answer? Submit a ticket to our help desk, and we'll get back to you. Within User Voice, "Contact Support" will bring up a form to submit a ticket.
